5. How to Setup the Software Kiosk Locally? (Step-by-Step Setup Guide)

To execute and run the Gramin-Nyaya interface on a local computer safely, execute the following steps:

  1. Clone Repo: Download the open-source repository from GitHub.
  2. Install Python: Ensure Python 3.10+ is installed on the host hardware.
  3. Install Dependencies: Run `pip install -r requirements.txt` to load backend components.
  4. Initialize Ollama: Install Ollama locally and download the required small models.
  5. Run App: Start the local server `python gramin_nyaya_main.py` and access the page.
